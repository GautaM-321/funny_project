## How It Works

1. The game presents the question "Will You Be My Valentine?"
2. Two buttons appear: "Yes! ğŸ’–" and "No"
3. When "No" is clicked:
   - A cute/funny message appears
   - The "Yes" button gets progressively bigger
   - The "No" button shakes
   - This continues with different messages each time
4. When "Yes" is clicked:
   - A success message appears
   - Floating hearts animation plays
   - Celebration emojis are displayed
